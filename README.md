# chess3x3
This project plays a 3x3 chess game

1. Firstly print the chess board:
   |   |   
---|---|---
   |   |   
---|---|---
   |   |   
2. Then player gaives a position 1 1/1 2/1 3 (x,y) for each position of the board
   '*' player
   '#' computer

3. if the row or col or diagonal is all the same
   '*' the player win
   '#' the computer win
